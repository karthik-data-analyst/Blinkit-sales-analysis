# Blinkit Sales Analysis

## Project Overview
This project analyzes Blinkit grocery sales data using Python in Google Colab.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Analysis
- Sales performance analysis
- Outlet type comparison
- Item category insights
- Data visualization
- Customer and product trends

## Outcome
Generated meaningful business insights from raw sales data using exploratory data analysis techniques.
